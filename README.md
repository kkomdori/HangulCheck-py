# HangulCheck-py

현재 input mode 가 한글임을 화면 가장자리에 작게 표시해주는 윈도우 응용프로그램입니다.  
input mode 가 영어일 때는 표시가 사라집니다.  

## 기능

- 위치/크기 설정
- 텍스트 정렬
- 텍스트색 선택
- 패널색 선택
- 투명도 설정

## 윈도우 시작 시 자동실행 방법 (스케줄러 등록)

#### 1단계: 작업 스케줄러 실행
- 키보드의 Win + R 키를 눌러 실행 창을 엽니다.  
- taskschd.msc 를 입력하고 엔터를 칩니다. (또는 윈도우 검색창에 "작업 스케줄러" 검색)

#### 2단계: 새 작업 만들기
- 우측 작업 창에서 **[작업 만들기]**를 클릭합니다. (주의: '기본 작업 만들기'가 아닙니다.)

#### 3단계: 일반(General) 탭 설정 (가장 중요)
- 이름(M): 작업 이름을 입력합니다. (예: HangulIndicator)
- 하단의 [가장 높은 수준의 권한으로 실행(I)] 체크박스를 반드시 체크합니다.
- 구성 대상(C): 현재 사용 중인 윈도우 버전(Windows 10 또는 Windows 11 등)으로 변경합니다.

#### 4단계: 트리거(Triggers) 탭 설정
- [새로 만들기(N)] 버튼을 클릭합니다.
- 작업 시작(G): 드롭다운 메뉴에서 **[로그온 할 때]**를 선택합니다.
- 설정(S): [모든 사용자] 혹은 [특정 사용자]를 선택합니다. (보통은 그대로 두셔도 됩니다.)
- (선택 사항) 만약 부팅 직후 인터넷 연결이 필요하거나 시스템이 안정화된 후 켜져야 한다면 **[작업 지연 시간(K)]**을 체크하고 '30초' 등으로 설정합니다.
- 확인을 누릅니다.

#### 5단계: 동작(Actions) 탭 설정
- [새로 만들기(N)] 버튼을 클릭합니다.
- 동작(I): [프로그램 시작] 상태에서
- 프로그램/스크립트(P): [찾아보기]를 눌러 실행할 응용프로그램(.exe)을 선택합니다.

  ★ 중요 팁 (시작 위치):
  일부 프로그램은 실행 파일이 있는 폴더 위치를 인식해야 오류가 안 납니다.
  [시작 위치(옵션)(T)] 란에 해당 실행 파일이 들어있는 폴더 경로를 복사해서 붙여넣으세요. (따옴표 없이 경로만 입력)

#### 6단계: 조건(Conditions) 탭 설정 (노트북 필수)
- 전원(Power) 항목: [컴퓨터의 AC 전원이 켜져 있는 경우에만 작업 시작] 체크를 해제합니다.
- 자동으로 그 밑의 [배터리 전원으로 전환되면 중지] 체크도 해제됩니다.

#### 7단계: 설정(Settings) 탭 설정 (꺼짐 방지)
- 이 부분을 놓치면 프로그램이 3일 뒤에 강제로 꺼질 수 있습니다.
- [다음 시간 이상 작업이 실행되면 중지(K)] 체크를 해제합니다. (기본값이 '3일'로 되어 있어 장시간 켜두는 프로그램이 강제 종료될 수 있습니다.)
- [요청 시 작업이 실행되도록 허용(R)]은 체크 상태로 둡니다. (테스트를 위해 필요)
- [작업이 이미 실행 중이면 다음 규칙 적용]: [새 인스턴스 실행 안 함] (기본값) 유지.

#### 8단계: 저장 및 테스트
- 모든 설정이 끝났으면 하단의 [확인] 버튼을 눌러 작업을 저장합니다.
- 작업 스케줄러 목록(가운데 창)에서 방금 만든 작업을 찾습니다.
- 마우스 우클릭 후 **[실행(R)]**을 눌러보세요.
- 프로그램이 관리자 권한 묻는 창(UAC) 없이 깔끔하게 실행된다면 성공입니다.
